Loop Output Explanation 
n = 2;
while (n < 10) 
{ 
   System.out.print(n + " "); 
   n = 2 * n; 
}
System.out.print(n);
 2 4 8 16 Note that 16 is printed after the loop has completed. 
n = 2;
while (n < 10) 
{ 
   n = 2 * n; 
   System.out.print(n + " "); 
}
System.out.print(n);
 4 8 16 16 In the third iteration, n is set to 16, and that value is printed. It is printed again after the loop has completed. 
n = 1;
while (n < 100) 
{ 
   System.out.print(n + " "); 
   n = 10 * n; 
}
 1 10 n is set to 100 in the third loop iteration. The loop exits because the condition n < 100 is no longer true. 
n = 1;
while (n <= 100) 
{ 
   System.out.print(n + " "); 
   n = 10 * n; 
}
 1 10 100 n is set to 100 in the third loop iteration. The loop executes one more time because the condition n <= 100 is still true. When n is set to 1000, the loop exits. 
n = 1;
while (n >= 100) 
{ 
   System.out.print(n + " "); 
   n = 10 * n; 
}
System.out.print("Done");
 Done This loop is never entered because the condition n >= 100 is false at the outset. 
n = 1;
while (n <= 100) 
{ 
   System.out.print(n + " "); 
}
System.out.print("Done");
 1 1 1 ... This is an infinite loop. n is never changed.
